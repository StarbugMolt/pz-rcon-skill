# Mod Item List Template

Copy this file per mod using this naming convention:

`mod-<modname>-items.md`

Examples:
- `mod-ki5-items.md`
- `mod-brita-items.md`

## Header
- Mod Name:
- Workshop ID:
- Mod ID:
- Enabled on server: yes/no

## Item Script IDs (RCON-ready)
- `ModName.ItemName`
- `ModName.OtherItem`

## Vehicle Script IDs (if any)
- `ModName.VehicleScript`

## Notes
- Only keep entries that are verified working on this server build.
- Keep one file per mod so activation/deactivation is clean.
