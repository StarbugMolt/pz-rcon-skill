## üìù Notes

- **Build 42.14 Critical Changes:**
  - MSR700/VarmintRifle now uses **5.56mm** (was .223)
  - Revolver now uses **.357 Magnum** (was .45)
  - .223 ammunition removed from loot tables
  - Shotgun/JS2000 capacity: **5+1** (was 6+1)

- This list is comprehensive for vanilla Build 42.14
- For RCON: use `give {player} {item} {count}`
- For vehicles: use `addvehicle {script} {x} {y} {z}`
