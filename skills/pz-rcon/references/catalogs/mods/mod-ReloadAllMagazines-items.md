# ReloadAllMagazines - Magazine Weapons

This mod adds weapons with detachable magazines.

## Notable Weapons (for RCON)
Use vanilla weapon names + magazine variants:

| Weapon | Code |
|--------|------|
| Pistol | Base.Pistol |
| Shotgun | Base.Shotgun |
| Hunting Rifle | Base.HuntingRifle |
| Assault Rifle | Base.AssaultRifle |

## Notes
- Magazine-based reload system
- Ammo types: Base.9mmClip, Base.ShotgunShells, Base.308Clip, Base.223Clip

## Use in Rewards
- Ammo drops for magazine weapons
- Full magazines as rewards
