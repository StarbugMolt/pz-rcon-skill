# PROJECTRVInterior42 - Walkable Vehicle Interiors

This mod adds walkable interiors to certain vehicles (RV-style).

## Vehicle Scripts
Uses vanilla vehicle scripts - mod adds interior meshes, not new vehicles.

### Available via RCON (vanilla + this mod):
- `Base.CarNormal`
- `Base.CarStationWagon`
- `Base.PickUpVan`
- `Base.Van`
- `Base.VanAmbulance`
- `Base.ModernCar`

### Notes
- Some vehicles require V key to enter (radial menu)
- Can be entered from trunk/back on some models
- Mod must be first in load order

## Use Cases
- Vehicle spawn rewards after events
- "Military convoy" narrative drops
- "Found abandoned RV" story beats
