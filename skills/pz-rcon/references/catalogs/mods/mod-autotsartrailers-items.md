# autotsartrailers - Trailers & Caravans

Mod adds various trailer and caravan vehicles.

## Vehicle Scripts (for RCON)

| Script | Description |
|--------|-------------|
| `autotsar.flatbed` | Flatbed trailer |
| `autotsar.cargotrailer` | Cargo trailer |
| `autotsar.hometrailer` | Home/caravan trailer |
| `autotsar.boattrailer` | Boat trailer |

## Requirements
- Requires `tsarslib` (Tsar's Common Library)

## Use Cases
- "Found an abandoned caravan" rewards
- Storage expansion story beats
- Mobile base narrative ("Found a trailer I can live in")
