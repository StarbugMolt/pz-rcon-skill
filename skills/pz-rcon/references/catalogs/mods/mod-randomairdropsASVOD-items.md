# randomairdropsASVOD - Random Airdrops

This mod adds random airdrops to the game world.

## Narrative Use
SIMON can reference airdrops in radio chatter:
- "Heard a plane overhead... might be an airdrop somewhere north"
- "Military's still running supply runs. Keep your eyes peeled"

## Notes
- Airdrops are spawned by the mod automatically based on config
- Cannot trigger via RCON directly (mod feature)
- Use narrative mentions to build anticipation
- Great for "I saw a drop nearby!" roleplay
